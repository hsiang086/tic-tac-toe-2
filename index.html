Game Rules

Tic Tac Toe² is an advanced version of the classic Tic Tac Toe game. Each cell of the main board contains a mini Tic Tac Toe board.

  • The game starts with Player X making a move in any of the 81 empty spots.
  • The next player's move must be in the mini-board corresponding to the cell of the previous move.
  • If a mini-board is won or ends in a draw, that board is closed, and the next player can choose any other open mini-board.
  • The first player to win three mini-boards in a row (horizontal, vertical, or diagonal) wins the game.
  • If all mini-boards are filled and no player has won, the game ends in a draw.
